介绍

BTrace是Java平台的安全，动态跟踪工具。

BTrace可用于动态跟踪正在运行的Java程序（类似于OpenSolaris应用程序和操作系统的DTrace）。 BTrace动态检测目标应用程序的类以注入跟踪代码（“字节码跟踪”）

BTrace是检查和解决线上的问题的利器，BTrace 可以通过编写脚本的方式，在服务不用重启的情况下，获取程序执行过程中相关信息。
